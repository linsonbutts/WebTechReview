XML: XML stands for extensive markup language which is used to describe data

DTD: Dtd stands for document type definition which defines the tags and elements used in xml or html

XSLT: XSLT stands for Extensible Stylesheet Language Transformations which is a language for transforming XML documents into other XML documents, or other formats such as HTML for web pages, plain text or XSL Formatting Objects, which may subsequently be converted to other formats, such as PDF, PostScript and PNG.

HTML:Html stands for hypertext Markup Language which is the standard markup language for documents designed to be displayed in a web browser.

CSS: Css stands for Cascading Style Sheet which are used to format the layout of Web pages

Web Forms: Web forms are an online page that allows for user input. It is an interactive page that mimics a paper document or form, where users fill out particular fields.

JavaScript: JavaScript is a programming language commonly used in web development.

DOM: The Document Object Model (DOM) is an application programming interface (API) for HTML and XML documents. It defines the logical structure of documents and the way a document is accessed and manipulated

DHTML:Dynamic HTML or DHTML refers to a combination of regular HTML (Hypertext Markup Language) and various programming languages to increase a web page's interactivity.

PHP: PHP. Stands for "Hypertext Preprocessor." (It is a recursive acronym, if you can understand what that means.) PHP is an HTML-embedded Web scripting language. This means PHP code can be inserted into the HTML of a Web page. ... This make PHP pages secure enough to access databases and other secure information.

ASP & ASP.NET: Search ResultsFeatured snippet from the web ASP.NET is an open-source server-side web-application framework designed for web development to produce dynamic web pages developed by Microsoft to allow programmers to build dynamic web sites, applications and services.

JSP:JavaServer Pages (JSP) is a collection of technologies that helps software developers create dynamically generated web pages based on HTML, XML, SOAP, or other document types.

Bootstrap: Bootstrapping describes a process that automatically loads and executes commands.

MySQL: MySQL is an Oracle-backed open source relational database management system (RDBMS) based on Structured Query Language (SQL).

HTTP:HTTP means HyperText Transfer Protocol. HTTP is the underlying protocol used by the World Wide Web and this protocol defines how messages are formatted and transmitted, and what actions Web servers and browsers should take in response to various commands.

UDP: User Datagram Protocol (UDP) is part of the Internet Protocol suite used by programs running on different computers on a network. UDP is used to send short messages called datagrams but overall, it is an unreliable, connectionless protocol.

TCP: TCP (Transmission Control Protocol) is a standard that defines how to establish and maintain a network conversation through which application programs can exchange data.

URN: A type of URI that identifies resource by name in a namespace.

IP: Internet Protocol is what designates the rules of how computers communicate on the interenet with each other.

IP Address: A necessary address that is unique to a that allows it to communicate over the internet.

FQDN: Fully Qualified Domain Name is made by writing a domain name with the hostname and domain name, this is essentially an absolute path to a host.

ARPANET: Advanced Research Projects Agenct Network was the precursor to the internet of today a communications system that was intended to link Pentagon research facilities together over telephone lines.

Web Server: Cloud based computers that receicve requests from web clients and formulates a response and sends this data back to the browser.

Port: Numbers of a 16 bit unsigned integer that is used to distinguish between various services being done over TCP,UDP,DCCP, and SCTP.

Web Hosting: A service necessary for publishing a website, A web-host is an IP-address that points to a specific computer that holds all the pages of a website and makes them available on the internet.

FTP: File Transfer Protocol is a protocol made to exchange files across the Internet.

SFTP: SFTP or Secure File Transfer protocol is a secure variant of the File Transfer Protocol, it transfers data on an SSH(Secure Shell).

SSH: Shell or Secure Socket Shell is a network protocol that provides users with a secure way to access a computer over an otherwise unsecured network.

API: Application Programming Interface are routines,tools and protocols used for the purpose of building software applications.

RESTful API: A RESTful API is an application programming interface that utilizes HTTP requests to affect data.

SOAP: Simple Object Access Protocol is a message protocol that makes it so various elements of an application to communicate with each other.

Requests: A demand from a system to receive a response over a protocol from another machine or host.

Response: The providied reply from a host or machine in response to having receiving a request.

HTTP Requests: A request done over Hyper Text Transfer Protocol for resources on the internet. TCP manages channels that responds to HTTP requests.

JPEG: Joint Photographic Experts Group A format for compressed image files supports up to 16 million colors in 8 bits.

PNG: Portable Network Graphic another format for compressed image files. Makes use off lossless compression so blurring is avoided with these.

Header tag: The header tag element represents a container for introductory content or a set of navigational links.

Footer tag: The footer tag defines a footer for a document or section.

Nav tag: The Nav tag defines a set of navigation links.

Article tag: The article tag specifies independent, self-contained content.

Main tag: The main tag specifies the main content of a document.

Aside tag:The aside tag defines some content aside from the content it is placed in.

Hr tag: The hr tag defines a thematic break in an HTML page.

A tag: The a tag defines a hyperlink, which is used to link from one page to another.

Link tag: The link tag defines a link between a document and an external resource and is used to link to external style sheets.

Img tag: The img tag defines an image in an HTML page and it has two required attributes: src and alt.

Figure tag: The figure tag specifies self-contained content, like illustrations, diagrams, photos, code listings, etc.

FigCaption tag: The figcaption tag defines a caption for a figure element.

Video tag: The video tag specifies video, such as a movie clip or other video streams.

Audio tag: The audio tag defines sound, such as music or other audio streams.

Object tag: The object tag defines an embedded object within an HTML document. Use this element to embed multimedia (like audio, video, Java applets, ActiveX, PDF, and Flash) in your web pages.

Param tag: The param tag is used to define parameters for plugins embedded with an object element.

Embed tag: The embed tag defines a container for an external application or interactive content (a plug-in).

Table tag: The table tag defines an HTML table.

Tr tag: tag defines an HTML table.

Th tag:The th tag defines a header cell in an HTML table.

Td Tag: The td tag defines a standard cell in an HTML table.

Caption Tag: The caption tag defines a table caption.

Thead Tag: The thead tag is used to group header content in an HTML table.

Tbody tag: The tbody tag is used to group the body content in an HTML table.

ColGroup tag: The ColGroup is a tag that specifies a group of one or more columns in a table for formatting.

Col tag: The col tag specifies column properties for each column within a colgroup element.

T foot tag: The t foot tag is used to group footer content in an HTML table.

Styletag: The styletag is a inline tag that specifies certain styles for the web page.

Script tag: The script tag is used to define a client-side script (JavaScript).

Canvas tag: The Canvas tag is used to draw graphics, on the fly, via scripting (usually JavaScript).

SVG tag The SVG tag defines a container for SVG graphics.
