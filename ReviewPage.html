URN: A type of URI that identifies resource by name in a namespace.

IP: Internet Protocol is what designates the rules of how computers communicate on the interenet with each other.

IP Address: A necessary address that is unique to a that allows it to communicate over the internet.

FQDN: Fully Qualified Domain Name is made by writing a domain name with the hostname and domain name, this is essentially an absolute path to a host.

ARPANET: Advanced Research Projects Agenct Network was the precursor to the internet of today a communications system that was intended to link Pentagon research facilities together over telephone lines.

Web Server: Cloud based computers that receicve requests from web clients and formulates a response and sends this data back to the browser.

Port: Numbers of a 16 bit unsigned integer that is used to distinguish between various services being done over TCP,UDP,DCCP, and SCTP.

Web Hosting: A service necessary for publishing a website, A web-host is an IP-address that points to a specific computer that holds all the pages of a website and makes them available on the internet.

FTP: File Transfer Protocol is a protocol made to exchange files across the Internet.

SFTP: SFTP or Secure File Transfer protocol is a secure variant of the File Transfer Protocol, it transfers data on an SSH(Secure Shell).

SSH: Shell or Secure Socket Shell is a network protocol that provides users with a secure way to access a computer over an otherwise unsecured network.

API: Application Programming Interface are routines,tools and protocols used for the purpose of building software applications.

RESTful API: A RESTful API is an application programming interface that utilizes HTTP requests to affect data.

SOAP: Simple Object Access Protocol is a message protocol that makes it so various elements of an application to communicate with each other.

Requests: A demand from a system to receive a response over a protocol from another machine or host.

Response: The providied reply from a host or machine in response to having receiving a request.

HTTP Requests: A request done over Hyper Text Transfer Protocol for resources on the internet. TCP manages channels that responds to HTTP requests.

JPEG: Joint Photographic Experts Group A format for compressed image files supports up to 16 million colors in 8 bits.

PNG: Portable Network Graphic another format for compressed image files. Makes use off lossless compression so blurring is avoided with these.

HTML Tag: Provides the root of an HTML document

Head Tag: provides information about the html document. Tags such as title,style, and link are stored here.

Title Tag: a necessary tag that provides for an HTML document.

Meta Tag: This tag gives metadata about the html document. This data is not visible on the page itself, and is often only used to provide info such as authors and page descriptions.

Body Tag: Defines an HTML document's body. All contents that are inteded to be displayed on the page such as links,images,and tables are inside the body.

P Tag: Simply defines an area for text or a paragraph.

H1-H6 Tags: Provides headings to the text ranked from most important to least.

Blockquote Tag: Provides a section for a quote from another source.

Address Tag: Provides the contact information for an author of a page,article, or document.

Pre Tag: The pre tag is used to display text that is in predetermined format, may be useful for displaying literature.

Em Tag: Simply emphasizes text.

Strong Tag: A tag that defines what is important text.

Q Tag: This tag provides a format for a brief quotation

Cite Tag: This tag is used to define the title of a cited source.

DFN Tag: A tag that indicates a defining instance of a term in other words it explains to users the uses of a term.

Code Tag: Provides a space for code from a programming language to be used in a document.

Mark Tag: Provides highliting for enclosed text.

Del Tag: Used to show what text has been deleted from a document.

Ins Tag: Displays what text has been inserted from another source into a document.

Abbr Tag: Defines an acronym or abbreviation used in a document.

Span Tag: Used to define a section within a document, groups inline (left right adjacent) elements and creates no visual changes on its own.

Br Tag: Provides a line break within text.
